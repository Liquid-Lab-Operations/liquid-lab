## ✅ Regras de Uso

✅ **Sempre use os arquivos oficiais** — nunca crie variações não autorizadas  
✅ **Para web/digital**: Use PNG com fundo transparente  
✅ **Para impressão**: Use vetor CMYK  
✅ **Mantém proporção**: Nunca redimensione desproporcionalmente  
✅ **Respeita cores**: Não invente novas cores ou variações  
✅ **Sem rotação**: Logo sempre na horizontal
